常规设置

配置网站基本信息和管理员信息，通过管理员信息的配置，可以用来接收网站的相关动态。

常规选项

  • 网站名称
  • 网站标题
  • 副标题
  • 网站域名
    请用http（或https）开头的域名，结尾不要包含“/”。
    例如：http://jpress.io。

管理员

  • 邮箱
  • 手机
  • 微信号

可选

  • 版权信息
  • 网站ICP备案号
