云端AI过滤

阿里云请参考 https://help.aliyun.com/document_detail/53427.html
腾讯云请参考 https://cloud.tencent.com/product/tms
小花儿AI请参考 https://market.aliyun.com/products/57124001/cmapi029968.html
泰岳语义工厂请参考 https://market.aliyun.com/products/57124001/cmapi033064.html

阿里云regionId请参考 regionId参数列表
腾讯云region请参考 region公共参数

阿里云和腾讯云服务商要配置参数：appId，appSecret，regionId； 小花儿AI和泰岳语义工厂服务商要配置参数：appCode

App Code 的认知方式为阿里云市场的认证方式，倘若通过 AppId 和 AppSecret 认证，就没必要再设置 App Code 认证了。反过来，若填写了 App Code 就没必要填写 AppId 和 AppSecret 了。
