网页链接样式设置

建议不要频繁改变此设置，频繁改变会影响到搜索引擎对网站内容的收录。

内容详情设置

  • 默认为“c”（content的首字母）
  • http://www.xxx.com/c?id=123
  • http://www.xxx.com/c?slug=press
  • http://www.xxx.com/c/123
  • http://www.xxx.com/c/jpress
  • http://www.xxx.com/20160501/123
  • http://www.xxx.com/20160501/jpress
  • http://www.xxx.com/article/123
  • http://www.xxx.com/article/jpress

伪静态

若 以上 设置为【动态】类型，此项设置无效。

  • 后缀
