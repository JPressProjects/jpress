## JPress 简介

JPress 是一个使用 Java 开发的、`开源免费` 的建站神器，灵感来源于 WordPress，目前已经有超过 10w+ 的网站使用 JPress 搭建，其中包括多个政府机构，200+上市公司，中科院、红十字会等。
