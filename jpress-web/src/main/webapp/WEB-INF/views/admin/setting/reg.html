登录注册 Register And Login

用户登录

不填写时，当用户登录成功后会跳转到用户中心 /ucenter 。

微信的 AppKey 和 AppSecret 请去 https://open.weixin.qq.com 申请。

QQ 的 AppKey 和 AppSecret 请去 https://connect.qq.com 申请。

微博 的 AppKey 和 AppSecret 请去 https://open.weibo.com 申请。

GitHub 的 AppKey 和 AppSecret 请去 https://github.com/settings/applications/new 申请。

码云 的 AppKey 和 AppSecret 请去 https://gitee.com/oauth/applications 申请。

钉钉 的 AppKey 和 AppSecret 请去 https://open-dev.dingtalk.com 申请。
