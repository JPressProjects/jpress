欢迎使用JPress。在开始前，JPress需要您提供如下信息。

1、数据库名
2、数据库用户名
3、数据库密码
4、数据库主机

我们会使用这些信息来创建一个名为 jboot.properties 的数据库信息文件。 如果自动创建未能成功，不用担心，您可以在文本编辑器中打开 jboot-sample.properties ，填入数据库信息，并将其另存为 jboot.properties 。

绝大多数时候，您的网站服务商会给您这些信息。如果您没有这些信息，在继续安装JPress之前您将需要联系他们。

如果您准备好了，那么，愉快的玩耍吧... 若需要更多帮助，您可以请访问JPress官网获取。
